Quick FAQ

  • Paano kung nakalimutan ang password? — Gumamit ng Forgot Password at i-check ang email/phone na naka-link.
  • Walang access sa email/phone? — Subukang i-recover muna ang email/number; o gamitin ang trusted contacts kung available.
  • Hacked ang account? — I-secure agad: palitan ang password, i-check ang sessions, at i-enable 2FA.

Note: Hindi kami official ng Meta. Guidance lang at pag-forward ng issue via email.

Email Delivery Setup

May dalawang paraan para ma-receive mo ang submissions:

  1. Direct via EmailJS (recommended) — Edit mo ang mga EMAILJS_* keys sa ibaba. Libre ang basic tier.
    • Gumawa ng account sa EmailJS, create Service + Template, then ilagay ang PUBLIC KEY, SERVICE ID, at TEMPLATE ID.
    • Pag naka-setup, automatic na magse-send ng email sa iyo sa bawat submit.
  2. Fallback Mailto — Kung walang keys, magbubukas ng email client gamit ang mailto: pre-filled. I-click ang “Test Mailto” para subukan.

Walang backend required; client-side lang ito. Para sa attachments, kailangan ng backend/server.
